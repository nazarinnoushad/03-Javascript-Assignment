6.	Create a BMI Calculator 
BMI = Weight(kg) / (height(m) * height(m))

```js
let weight = prompt ("weight (in Kg)")
let height = prompt ("height (in m)")
BMI = weight / (height * height) 
let message = ("your BMI")
alert(`${message} ${Math.round(BMI)}`)
```
